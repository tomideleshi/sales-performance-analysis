# Sales Performance Analysis

## Project Overview
An end-to-end exploratory data analysis (EDA) project analyzing retail sales performance across salespeople, regions, and product categories using Python.

## Tools Used
- Python (Google Colab)
- Pandas
- Seaborn
- Matplotlib

## Key Features
- Data wrangling and aggregation using Pandas groupby
- Professional bar charts and visualizations using Seaborn and Matplotlib
- Business insights summary with actionable recommendations
- Analysis covers 5 salespeople, 4 regions, and 5 product categories

## Key Insights
- Top performing salesperson: Diana ($365K revenue)
- Strongest region: West ($468K total sales)
- Best performing product: Laptop ($423K revenue)
- South region underperforming by $138K compared to West
